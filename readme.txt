== Description ==

WP Smart FAQ Plugin used for quickly and easily add faq to website. With the plugin installation a Custom Post Type named "FAQ" will be created automatically. 

To show all FAQ's items a shortcode [faq] needed, just create a page for your FAQ section and update the page with shortcode [faq]. If you want to show FAQ based on category. 

Go To --> FAQ Category , Create a category and To show this category used [faq id="category id"] i.e [faq id="8"] where 8 is category id. 

= Features of Plugins =
* WP Smart FAQ Plugin includes- Title, Description.
* Categary based FAQ

== Installation ==

To install WP Smart FAQ using the built-in plugin installer:

    1. Go to Plugins > Add New.
    2. Type in WP Smart FAQ in Search Plugins box or click a tag link below the screen.
    3. find the WP Smart FAQ Plugin in the list.
    4. Click Install Now to install the WordPress Plugin.


To install WP Smart FAQ manually:

   1. Download your WordPress Plugin to your desktop.
   2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
   3. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
   4. Go to Plugins screen and find the rDuplicator Plugin in the list.
   5. Click Activate to activate it.

* Create a new page named it "FAQ" or something. In the Editor place the shortcode [faq]
* To show all FAQ's items a shortcode [faq] needed, 
* To show FAQ category wise used Shotcodes like [faq id="6"], where is is category id

Yes, all registered custom post types are supported.

= How to display FAQs in in page/posts? =

Shotcodes like [faq] used to display FAQs in page/posts, See description and installation for more details.
